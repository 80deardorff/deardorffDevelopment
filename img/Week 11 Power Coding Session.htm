Exercises

Important: Make sure to add comments to your code to help your future self.

  1. Create a copy of your template folder and rename it to week11exercise.
  2. Drag this new folder into your 'week11' folder inside your 'codify' folder.
  3. When working through these exercises, there are bits and pieces of the instructions that are deliberately left out. You should start thinking about what you THINK you will need in order to accomplish the request.
  4. In HTML: add an image and two buttons. (reference #5 for the content for your buttons)
  5. In jQuery: make the image element appear and disappear when you click the corresponding button.
  6. Create a third button which does the work of BOTH the previous buttons (i.e. when the image is visible and you click the button, the image disppears; when the image is not visible and you click the button, the image reappears). Open your inspector and notice HOW the image is being hidden/unhidden.
  7. In CSS: give your image a transition property with a value of "0.3s ease".
  8. In CSS: create a class selector for a class called "fade" with a property of "opacity: 0".
  9. Create a fourth button which toggles this new "fade" class on and off for the image element when clicked.
  10. In jQuery: make the image change to a different image on hover (think about what HTML attribute commands the content of an image element).
  11. In HTML: create an unordered-list with two list items (make sure these li's have some text content). Then create a button below this unordered list.
  12. In jQuery: when this button is clicked, add a new list-item with some text content to this unordered list.
  13. Finally, in HTML: create one last button.
  14. In jQuery: when this button is clicked, cause an element to render with a red background, which covers your entire window and all of the elements on your page. Make sure that if you try to scroll, you can still see nothing but the red background.

Once you have completed these exercises, try these for an added challenge.

  1. In HTML: create an h1 below your image and buttons.
  2. In jQuery: make the text content of this h1 change to reflect whether or not the "fade" class is active on the image (i.e. make the text change between "fade on" and "fade off" or something similar).
  3. In #10 above, you made the image change to a new image on hover. Now can you make it change BACK when you exit the hover (when you move your mouse off of the image element)?
  4. In jQuery: cause each list-item to change color, font-size, and font-family on hover (HINT: you might need to do a double-take on this one, if ALL of your list-items are changing when you hover on any ONE of them, you'll need to think carefully about how to make sure only this one which you're hovering over will change). Don't forget to make them change BACK when you exit the hover.

Steps For Making the Animated Full-Page Menu jQuery Project - remember to break it down into manageable steps:

  1. Paper prototype your site.
  2. Code your site in static HTML and CSS, including the menu to be animated. When this is done, your site should look exactly how you'd like when your menu is OPEN.
  3. Now add any classes or stylings you need to completely HIDE your menu and its content. In this way, we'll determine the starting state of our menu. For example, if you want your menu to slide down from the top of your window, you may want to force the menu to have a height of 0.
  4. Plan how you will use jQuery to change the styling of your menu to reach the OPEN end state. For example, if your menu has a height of 0, plan how to use jQuery to give it a height of 100vh.
  5. Write and test your jQuery. See if your menu opens and closes as you expect.
  6. Add CSS transition properties or incorporate animate.css classes to make your animated full-page menu even more interesting!

Steps For Making the Movie Rating List App - remember to break it down into manageable steps:

  1. Paper prototype your site.
  2. Code your site in static HTML and CSS with one or two sample movie titles and corresponding ratings. By this point, your site should look like you've already entered the names and ratings of two films.
  3. Now set up your function to add a static film name and rating to your site. Use jQuery to invoke your function with an event handler.
  4. Swap out the static film name and rating with the user-input film name and rating. Don't forget to use as much jQuery and as little vanilla JS as possible.
